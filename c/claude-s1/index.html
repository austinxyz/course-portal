常用 Q&A

按主题分组，受众 AI 新手——先结论、少术语。标 ⚠️ 的涉及隐私 / 费用 / 数据。

A. 账号 · 注册 · 费用

一定要付费吗？免费版够用吗？
免费版能体验对话；但今天演示的 Projects、连接器、Cowork、Dispatch 多属于付费档（Pro/Team）。想跟着搭完整流水线建议 Pro。先用免费熟悉，觉得有用再升级。
怎么注册？需要特殊网络吗？
claude.ai 注册，邮箱即可。部分地区访问受限，按你所在地的常规上网方式即可（课上第 7 页有注册步骤）。
手机能用吗？
能。手机 App 可对话，还能用 Dispatch 在手机上一句话触发电脑端流水线（第 14 页 + 花絮演示）。
怎么省 token / 不浪费额度？
token ≈ 你和 Claude 来回的字数，越省越耐用、付费档也更划算。几条实用习惯：
① 长上下文放 Project 文件   → 简历/方法论上传一次，别每次对话重复粘贴
② 一个任务开一个新对话      → 旧话题聊完就新建，避免历史越滚越长、每轮重读
③ 提问先给结论再给细节      → 说清"要什么、给谁、什么格式"，少来回澄清
④ 大文件只放需要的部分      → 几百页 PDF 截取相关段落，别整本塞进去
⑤ 复用提示词文件           → pipeline-prompts.md 直接调用，不临场重写长提示
记住：省 token 的核心不是少说，是别重复说——把固定信息沉淀到 Project，每次只发当次新增的那点。

B. Claude vs 其他工具

和 ChatGPT 有啥区别？已经用 GPT 了还要学这个吗？
对话能力两者都强。今天重点不是"哪个模型聪明"，而是 Cowork 的协作方式——Projects 存文件当长期上下文、连接器接 Gmail/Drive、Dispatch 手机触发、把提示词沉淀成可复用流水线。这套"系统化用法"是差异所在（第 7 页对比）。
中文支持好吗？
好。全程可中文对话与输出（演示里 Outreach 正文用英文是因为目标读者是英文招聘方，可按需调）。

C. Cowork · Projects · 文件

Project 是什么？和普通对话有啥区别？
Project = 带"项目文件夹 + 长期记忆"的工作区。放进去的文件（简历、方法论）每次对话自动带入，不用反复粘贴（第 10 页）。
记忆文件 / Custom Instructions 干嘛用？
Custom Instructions 是给这个 Project 定的"长期规矩"（比如触发词「检查新JD」要做什么）。配一次，之后每次自动按规矩走（第 12 页 + 过渡段演示）。
上传的文件 Claude 记多久？换对话还在吗？
在 Project 里的文件长期保留，跨对话都在；这正是它和一次性对话的区别。
记忆文件、CLAUDE.md、Skill 到底有啥区别？
一句话：记忆文件让 AI「懂你」，CLAUDE.md 让 AI「懂这个项目怎么干」，Skill 是 AI「随身带的一项能力」。
用「带新员工」打比方：
| | 是什么 | 作用范围 | 啥时候生效 | 能分享给别人吗 |
|---|---|---|---|---|
| 记忆文件（含 Custom Instructions） | 你的个人档案 + 公司手册（简历、5C 方法论、长期规矩） | 只在这个 Project 内 | 这个 Project 里每次对话自动带入 | 跟着 Project 走 |
| CLAUDE.md | 贴在某个工作文件夹门口的「这个项目怎么干」便签 | 某个本地文件夹内 | Cowork 一进这个文件夹自动读取 | 跟着文件夹走 |
| Skill | 员工随身的一项专业技能 / 工具包（含做法，可带脚本） | 跨所有 Project，全局可用 | 任务匹配到时才被调用（不是每次都加载） | ✅ 可单独打包、双击安装、发给朋友 |
- 记忆文件 vs CLAUDE.md：本质都是「记忆」。区别在放哪——记忆文件是你在 claude.ai 的 Project 里上传的；CLAUDE.md 是放进本地工作文件夹、Cowork 按文件名自动认的。新手先掌握记忆文件即可，CLAUDE.md 是进阶（用 Cowork 处理本地文件夹时才会碰到）。
- 记忆文件 vs Skill：记忆文件是「关于你 / 这个项目的信息」，每次都带；Skill 是「一项可复用的本事」，需要时才拿出来用，且能跨项目、能分享。今天把 4 个 Agent 提示词存进 pipeline-prompts.md，本质就是一个记忆文件（存在 Project 里、可复用）；触发词「检查新JD」时 Claude 去读它依次执行。再进一步打包成「求职流水线 Skill」，就能跨 Project 一行触发、发给别人用（第 13 / 20 页）。

D. 连接器 · 隐私安全 ⚠️

连 Gmail 安全吗？它会读我所有邮件吗？
连接器走 Google 官方授权（OAuth）。Claude 只在你明确要求时按那次任务去读（比如"找主题含 JD 的邮件"），不是后台常驻扫描。随时可在 Google 账号的"第三方访问"里撤销。
我的简历/数据会拿去训练模型吗？
⚠️ 如实说：企业/API 数据默认不用于训练；消费版（claude.ai）是否用于训练取决于你账号的隐私设置，建议进设置确认并按需关闭。课上我不打包票，以官方隐私设置为准。
连接器怎么撤销 / 取消授权？
两处都可：Claude 端连接器设置移除；或 Google 账号 → 安全 → 第三方应用访问里撤销。

E. 5C 流水线 · Demo

我不是工程师、也不在找工作，这套有用吗？
有。今天用"找工作"只是例子。核心是私有信息 + 领域知识 + 4 步流水线的模式——换成你的场景（销售线索、客户调研、内容生产……）和你的方法论文件，同一套就变成你的系统（第 18/19 页）。
提示词写不好怎么办？
这正是今天的核心心态：别想一次写完美。先写极简版能跑，再让 Claude 帮你补全成专业版，不满意就追问迭代。你的责任是"知道要什么"，不是背框架（Demo Agent 1/3 专门演示迭代）。
4 个 Agent 必须按顺序吗？
是。后一个读前一个的产出（如 Outreach 读 JD 分析 + 公司调研），所以按 1→2→3→4。顺序写在 pipeline-prompts.md 里，触发后自动执行。
触发词「检查新JD」能改吗？
能。它只是你在 Custom Instructions 里自定义的词，改成任何你顺口的都行。

F. Skills · Dispatch · 自动化

Skill 和提示词文件有啥区别？
提示词文件是"这个 Project 内"的复用资产；Skill 是打包好、跨项目一行触发的能力。今天先把流水线跑通，封装成 Skill 是延伸方向之一（第 20 页）。
Dispatch 是什么？手机怎么触发？
Dispatch 让你在手机上发一句话，触发电脑端 Cowork 跑任务（第 14 页）。花絮里演示了手机一句话自动批改作业。
能定时自动跑吗？
能，用 Schedule 设定时（如每天早上自动扫新 JD）。属课后延伸方向（第 19 页）。

课后怎么练？作业是什么？
作业：用 Cowork 搭一条你自己场景的流水线，截图发群（第 24 页）。下载包 + homework 指南里有完整材料。
有回放和材料吗？
材料：第 3 / 17 页的下载按钮（starter-project + 2 封 JD 邮件 + 提示词表）。回放：[课后补充安排]。

I. 零基础 / 本地文件整理（真实学员案例）

我 100% 小白、已退休，群里术语像"天书"，怕跟不上、达不到预期。
吃定心丸：课为零基础设计，带着一步步做，材料可下载、有回放和作业，不用当场全记住。术语都用大白话讲；会用电脑、会打字、会收发邮件就够。鼓励多问"笨问题"——没有笨问题。
在哪下载 Claude？用哪个版本？（MacBook）
- 最简单：浏览器开 claude.ai，邮箱注册即用，不装任何东西。
- Mac 桌面 App：claude.ai/download；手机：App Store 搜 "Claude"。
- 注册后默认最新版，不用选版本。免费版先体验，进阶功能（Projects/连接器）需 Pro。
Claude 基本特点 / 注意什么？
中文问即可。默认看不到你电脑里任何文件——只有你主动上传或授权连接时才看那一部分，随时可撤销。心态：别怕问、别想一次写对，追问会越帮越准。
⚠️ 能帮我整理 / 清理 / 缩减 Mac 上的 600GB 文件、照片、音视频吗？
如实讲清预期：Claude / Cowork 是云端助手，不能自己进你的 Mac 扫描、移动、删除本地文件，也无法靠"看"批量去重照片视频；自动清盘不是它的活，硬做有误删风险。
它能当整理顾问 + 私人教练：帮你定清理计划、一步步教你用 Mac 自带工具做、判断哪类文件能删/怎么转移省空间。真正动手靠 Mac 功能 + 小工具：
① 先备份         → 重要文件先拷外置硬盘 / iCloud（动手前第一步！）
② 看哪占地方     → 苹果菜单 → 关于本机 → 储存空间 → 管理
③ 照片/视频瘦身  → 「照片」开 iCloud「优化储存」；大视频转外置硬盘
④ 找重复/大文件  → 去重工具（如 Gemini）+「管理」里的建议
⑤ 清空废纸篓     → 删除后清空才真正释放空间
课上用法：把目标告诉 Claude，让它把这 5 步细化成"适合我这台 600GB MacBook Air 的清理清单"，照着做、边做边问。它陪你想、教你做，不替你自动清盘。
这样的小白，课前需要准备什么？
只两件：① claude.ai 注册好账号（免费版，能登录即可）；② 想跟做的话，记下自己设备现状（如"600GB MacBook Air，主要音视频+照片"），课上拿真实例子让 Claude 出方案。其余课上带做。
