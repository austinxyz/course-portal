常用 Q&A

按主题分组，受众 AI 新手——先结论、少术语。标 ⚠️ 的涉及费用 / 隐私 / 能力边界。

A. 安装 · 环境（课前作业）

Claude Code 装不上 / 命令报错？
群里发报错截图，我帮你看；需要的话课前另约时间弄好。命令：Win irm https://claude.ai/install.ps1 | iex；Mac curl -fsSL https://claude.ai/install.sh | bash。不用先装 Node。
一定要付费吗？
⚠️ Claude Code 跑课上演示需要 Claude Pro（$20/月）；Obsidian 完全免费。先订上，课后觉得不合适随时取消。
要装 VS Code 和 GitHub 吗？
不用——它们是选做。课上演示只用 Claude Code + Obsidian。想把知识库管理得更专业再装（参考资料页有链接）。
Windows 还是 Mac？
都行。命令分两套（见课前准备），其余操作一致。

B. Claude Code 是什么 · 和别的区别

Claude Code 和 ChatGPT / Cowork 有啥不同？
ChatGPT 在对话框里，聊完就结束；Claude Code 跑在你电脑上，能直接读写你硬盘里的文件、建系统。Cowork（第一节）跑流程、用完即走；Claude Code 建持久的本地资产。
今天要写代码吗？
不用。全程用中文跟 AI 说需求，它帮你建文件、搭知识库。Claude Code 名字里有 Code，但今天教的不写一行代码。

C. Obsidian · 知识库

Vault 是什么？会改我的文件吗？
Vault = 一个普通文件夹。Obsidian 只是换种方式打开它来看 Markdown，不会改动你的文件。
怎么打开我已有的知识库？
Obsidian → Open folder as vault → 选那个文件夹（不是 Create new vault）。课前准备里有图文。
Graph View 是干嘛的？
把笔记之间的 [[链接]] 画成关系图，一眼看清知识结构。左侧"圆点连线"图标打开。

D. 知识库内容 · wiki-extract

我没有专业领域 / 不知道做什么知识库？
跟着我做理财知识库就行（现成下载包 + 傻瓜教程）。有自己领域（保险/律师/医疗…）更好，换成你的内容照做。
wiki-extract 一定要联网搜资料吗？
不一定。你是领域专家，有自己的材料就直接喂给它提炼（粘贴 / 文件 / 链接）；没材料它才去搜。核心价值是"提炼"，不是替你瞎搜。
raw_material 和 wiki 有什么区别？
raw 是原始素材（带来源），wiki 是你消化后的结构化条目。想溯源去 raw 找。

E. Skills

Skill 是什么？
把一段结构化提示词 + 流程存成文件，下次一个命令复用。自己做的针对你的领域；社区的（superpowers / ppt-master）通用，按需组合。
superpowers / ppt-master 怎么装？
在 Claude Code 里：
/plugin marketplace add obra/superpowers → /plugin install superpowers@superpowers
/plugin marketplace add hugohe3/ppt-master → /plugin install ppt-master@ppt-master
ppt-master 是什么？
AI 生成可编辑 PPTX 的工具。今天这套课件就是它做的。

F. MCP · 连接器

MCP 和 Connector 有啥区别？
Connector 是官方预置、点几下就能用的集成（Gmail/Drive/日历）；MCP 是开放协议，能接几乎任何工具或自建服务，可扩展、需配置。不是"只读 vs 读写"——两者都能读能写。
Notion MCP 能干嘛？
让 Claude 直接在你的 Notion 里建记录、更新状态。属课后延伸方向。

G. 隐私 · 客户数据 ⚠️

客户的数据安全吗？会上传吗？
演示里客户档案放在本地 output/ 目录，加进 .gitignore 不进 Git、不外传。客户具体数字只在本地。wiki/ 只存通用知识、不含客户信息。
用 GitHub 备份会泄露客户信息吗？
有客户/隐私数据就建私有仓库（Private），或把这些文件放进 .gitignore 不上传。

H. 落地 · 赚钱

这套怎么帮我赚钱？
把你的专业知识结构化成知识库 → 用它更快、更稳地服务客户（出分析、出 PPT、答疑）。知识库是工具，服务客户才是变现点。
我不是理财顾问，也能用吗？
能。理财只是演示例子。保险、律师、医疗、教育、任何"靠专业知识服务人"的行业，换成你的领域照做。

作业交什么、交到哪？
选你自己的领域：建 3 篇 wiki（先建 /wiki-extract 再用）→ 用 superpowers 做客制化方案 → ppt-master 出 PPT。发邮件到 austin.aicourse@gmail.com，主题 【Session 2 作业】+ 你的名字，附 3 篇 wiki 片段 + PPT 截图 + 200 字心得。

课上先看懂"为什么这么做"，材料和脚本课后能自己重跑；作业群里交，不强求当场完成。
